# 概要

Zenn.dev に記事を共有するためのレポジトリ

自分のプロフィールは[こちら](https://zenn.dev/def_yuisato)

# Zenn CLI

記事作成

```shell
$ npx zenn new:article
```

プレビュー

```shell
$ npx zenn preview
```

プレビューの限定 URL を deploy する

```shell
$ ./preview-deploy.sh
```
